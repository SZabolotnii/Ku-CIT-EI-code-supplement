# Data Policy

This repository is a public code and generated-artifact supplement for a simulation-based manuscript.

## Included

- R scripts used for simulation diagnostics and classifier probes.
- Generated CSV tables used to verify reported Monte Carlo results.
- Generated manuscript-support figures.
- R session snapshots for the main follow-up runs.

## Excluded

- Raw BRFSS 2010 files.
- Processed BRFSS `.rds` files.
- Third-party source articles or PDFs.
- Private scratch files or local checkout metadata.

## BRFSS Boundary

BRFSS 2010 PHQ-8 material is referenced only as public-data context for the Wiedermann-Shi application setting. The manuscript results in this supplement are simulation-based, and the public supplement does not require redistribution of BRFSS data.
